Introduction to Web Design (CIS-224)

In CIS-224, Introduction to Web Design, we learn how to build Web pages from the ground up, using Hypertext Markup Language (HTML) and Cascading Style Sheets (CSS). As a final project, each student designs and builds his or her own Web site to showcase their skills.
